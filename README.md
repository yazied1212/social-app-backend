# Social App - Node.js Backend

## Description
Backend API for a simple social media app. Users can register, login, create posts, like, comment and chat. Built using Node.js, Express, and MongoDB.

## Technologies Used
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT for authentication
- bcrypt for password hashing

## Features
- User Registration & Login
- OTP verification for user authentication
- JWT Authentication
- CRUD operations for posts
- Comment and Like system
- Error handling and validation
